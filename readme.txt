STEPS THAT I HAVE FOLLOWED TO BUILD THIS PROJECT:

Commands (Windows PC):
1. Install the Ionic CLI globally:
npm install -g @ionic/cli

2. Create Project:
ionic start hope-trial-task tabs --type=angular

3. Move into the project folder:
cd hope-trial-task

4. Build the project for production
ionic build --prod

5. Run project in Android Studio (make sure android studio is installed in your PC):
ionic capacitor sync android
ionic capacitor open android

6. To test project in web browser:
ionic serve

-----------------
-----------------
TO USE THIS PROJECT:

1. clone the project
2. cd hope-trial-task
3. RUN Below Commands:
    a. npm install
    b. ionic build --prod
    c. ionic capacitor sync android
    d. ionic capacitor open android
4. To test in Browser, RUN:
    a. ionic serve
